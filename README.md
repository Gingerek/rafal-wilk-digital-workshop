# Rafal Wilk Digital Workshop

Static online version of the platform. Open `index.html` locally or publish the `site` folder with GitHub Pages, Netlify, or Cloudflare Pages.

## Structure

- `index.html` - platform shell and homepage
- `modules/` - extracted embedded applications
- `assets/` - reserved for extracted images and shared assets

## Privacy note

This is a static frontend. Do not commit real API keys or private data into HTML, CSS, or JavaScript files.
